# Security Policy

## Supported Versions

Only the latest release version on the `main` branch is actively supported with security updates.

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

We take the security of the UAV Digital Twin platform seriously. If you find any vulnerabilities or security risks, please do NOT create a public GitHub Issue.

Instead, report them by emailing the maintainer directly at **security@uavplatform.com**. We will acknowledge your report within 48 hours and coordinate a patch fix release.
